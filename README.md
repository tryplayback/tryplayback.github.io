# Playback

Landing page for **Playback** — a small desk display that shows the song, artist, runtime and album art of whatever's playing.

Static site, no build step. Open `index.html` or serve the folder:

```bash
npx serve .
```

## Structure
- `index.html` — markup (hero, the device, finishes carousel, the creator, footer)
- `styles.css` — design system + all styling
- `app.js` — interactions (finish carousel, parallax, reveals, form)
- `assets/` — product renders & headshot
